Virtual Local Area Network

[[LAN]]

Связаны даже если находятся в разных местах.
Позволяют на одном физическом порту роутера создавать несколько изолированных виртуальных сетей.
Тот же широковещательный [[Коммутатор (Switch)]] будет посылать только на порты, в одном VLAN с которым находится источник.

```
interface range fa0/1-5
switchport mode access
switchport access vlan 10
```
VLAN на свитче начинает подписывать трафик тэгами VID (4 байта в каждом фрейме)

Порт на коммутаторе в режиме Trunk кидает всё на другой коммутатор, который читает VID и в его сети рассылает в том же VLANе.
