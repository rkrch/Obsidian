**Domain Name System**

Сначала принимает Resolver - распознающий DNS сервер. Обычно у провайдера
Он обращается к Root, который посылает на TLD (Top Level Domain) и Authoritative nameserver. Ну и в кэш да

## Типы запросов
* Recursive (Рекурсивный) - ip
* Iterative (Итеративный) - ip или Authoritative DNS
* Inverse - доменное имя по ip
