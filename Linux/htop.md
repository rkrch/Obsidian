
### `htop` - улучшенный top (нужна установка)
```bash
sudo apt install htop
htop                      # цветной, с удобным управлением
```
**Преимущества htop:**
- Цветной интерфейс
- Управление мышью
- Процессы в дереве
- Быстрая фильтрация

 ### Горячие клавиши в htop:
	- `F9` - меню завершения процессов
	- `F3` - поиск
	- `F4` - фильтр
	- `Space` - отметить процесс

```
# Найти процессы Java
ps aux | grep java

# Мониторить конкретный процесс
top -p $(pgrep nginx)
```
