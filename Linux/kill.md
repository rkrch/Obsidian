## ⚡ `kill` - управление процессами

### Основное использование:
```bash
kill PID                  # вежливое завершение (SIGTERM)
kill -9 PID               # принудительное завершение (SIGKILL)
kill -15 PID              # то же что и kill PID
killall process_name      # завершить все процессы с именем
```

### Практические примеры:
```bash
# Найти и завершить процесс
ps aux | grep vim
kill 1234

# Принудительно завершить зависший процесс
kill -9 1234

# Завершить все процессы chrome
killall chrome
```
